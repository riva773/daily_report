4. attr_accessorとは何か？
## 解答
1. ```
   class Sample   
   end
   ```

2. インスタンス変数　インスタンスごとに固有の値を持つことができる。同じクラスの別メソッドから参照できる
3. newメソッドで作成されたオブジェクト
4. ゲッターとセッターを簡単に追加する機能。　クラス名の次の行に以下のような形で書く
```
class Sample
  attr_accessor :name,:age
```
